# Task Manager - JavaScript Application

## Description
A simple task manager web application built with pure JavaScript, HTML, and CSS. Users can add, complete, delete, and filter tasks. Tasks are saved in the browser's localStorage.

## Live Demo
View the live application here: [Task Manager](https://maxtinka.github.io/module2/)

## GitHub Repository
[https://github.com/maxtinka/module2](https://github.com/maxtinka/module2)

## Features
- Add new tasks
- Mark tasks as complete/incomplete
- Delete tasks
- Filter tasks by All, Active, or Completed
- Task counter showing remaining incomplete tasks
- Persistent storage (tasks survive page refresh)

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- localStorage API
