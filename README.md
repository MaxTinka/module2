# 📋 Task Manager – JavaScript Application

## Description
A feature-rich task manager web application built with pure JavaScript, HTML, and CSS. Users can create tasks with titles, descriptions, due dates, and times. Tasks can be marked complete, deleted, and filtered by status. All tasks are saved in the browser's localStorage for persistence across page refreshes.

## Features
- ➕ Add tasks with title, description, due date, and due time
- ✅ Click task title to mark as complete/incomplete
- 🗑️ Delete tasks
- 🔍 Filter tasks by: All, Active, or Completed
- 📊 Task counter showing remaining incomplete tasks
- 💾 Persistent storage using localStorage
- 🎨 Clean, responsive card-based UI with hover effects
- 📱 Mobile-friendly design

## Technologies Used
- **HTML5** – Structure and layout
- **CSS3** – Styling, gradients, and responsive design
- **JavaScript (ES6)** – DOM manipulation, event handling, array methods, and localStorage API
